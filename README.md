# Tutorial: Compressed Sensing using Generative Models (Bora et al., 2017)

This repo partially implements (in PyTorch) experiments of the paper 
*[Bora, A., Jalal, A., Price, E., & Dimakis, A. G. (2017, July). Compressed sensing using generative models. In International conference on machine learning (pp. 537-546). PMLR](https://proceedings.mlr.press/v70/bora17a/bora17a.pdf)*,
originally implemented in this <a href="https://github.com/AshishBora/csgm" target="_blank">repo</a>.

The only parts of the code to complete are in the notebook.

### Getting started

The code requires pixi: https://pixi.prefix.dev/latest/installation/.

Then, clone and enter the repo, and open the notebook running ```pixi run start``` in your terminal.
