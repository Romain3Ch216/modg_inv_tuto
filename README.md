# Tutorial: Compressed Sensing using Generative Models (Bora et al., 2017)

This repo partially implements (in PyTorch) experiments of the paper 
*Bora, A., Jalal, A., Price, E., & Dimakis, A. G. (2017, July). Compressed sensing using generative models. In International conference on machine learning (pp. 537-546). PMLR*,
originally implemented in this repo: <a href="https://github.com/AshishBora/csgm" target="_blank">https://github.com/AshishBora/csgm</a>.

The only parts of the code to complete are in the notebook.

### Getting started

The code requires pixi: https://pixi.prefix.dev/latest/installation/.

Then, simply open the notebook running:

```pixi run start```
